# ⚡ Glitch - Modern Online Games Platform

![Glitch Banner](https://img.shields.io/badge/Glitch-Online_Games-6C63FF?style=for-the-badge&logo=flutter)
![Flutter Web](https://img.shields.io/badge/Flutter-Web-02569B?style=for-the-badge&logo=flutter)
![Responsive](https://img.shields.io/badge/Responsive-Mobile_%26_Desktop-success?style=for-the-badge)

**Glitch** is a premium, high-performance online gaming platform built with Flutter. It offers a seamless, responsive experience across Web, Desktop, and Mobile devices, featuring a sleek modern design with full support for Dark Mode and multi-language localization.

[**🌐 View Live Demo**](https://kerolos77.github.io/Glitch/)

---

## ✨ Key Features

- **📱 Fully Responsive**: Optimized for every screen size using `flutter_screenutil`. Seamlessly switches between a Desktop Sidebar and a Mobile Drawer.
- **🌗 Dynamic Themes**: Premium Dark and Light modes with curated color palettes.
- **🌍 Multi-language Support**: Full support for **English** and **Arabic** (RTL) localization.
- **🎮 Built-in Games**: Includes a fully functional, high-performance **Minesweeper** game with adjustable difficulty.
- **🚀 Ultra-Fast Performance**: Built on Flutter's efficient rendering engine for a "Glit-free" experience.
- **🔒 Secure Web Features**: Custom context menus disabled for a native gaming feel.

---

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev/) (Channel Stable)
- **State Management**: [BLoC](https://pub.dev/packages/flutter_bloc) & [Provider](https://pub.dev/packages/provider)
- **Localization**: [Easy Localization](https://pub.dev/packages/easy_localization)
- **Responsiveness**: [Flutter ScreenUtil](https://pub.dev/packages/flutter_screenutil)
- **Typography**: [Google Fonts](https://pub.dev/packages/google_fonts) (Poppins)

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (latest stable version)
- Chrome or any modern web browser

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Kerolos77/Glitch.git
   cd Glitch
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the project:**
   ```bash
   flutter run -d chrome
   ```
